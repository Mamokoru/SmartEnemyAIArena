Smart Enemy AI Arena is a compact combat prototype showcasing advanced enemy behavior using a finite state machine (Patrol, Chase, Attack), NavMesh navigation, and a perception system with vision-based detection. Its key feature is adaptive AI: the enemy learns from successful attacks by remembering effective angles and prioritizing them in future engagements. This creates evolving, less predictable combat. The project demonstrates clean architecture, modular system design, and real-time decision-making, highlighting a shift from scripted enemies to adaptive, intelligent gameplay systems.
